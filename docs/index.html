Incentives & Analytics API (1.0.0)

Download OpenAPI specification:

Client-facing API for offers, incentives, trackers, and retail performance metrics.

Authentication

  • All endpoints require a Bearer token.

Offers

Get current offers

Authorizations:
bearerAuth
query Parameters
dealerId
required
string
Example: dealerId=DLR-004219

Dealer identifier.

market
string
Default: "US"
Enum: "US" "CA"

Market/country context for content and eligibility.

locale
string
Default: "en-US"
Example: locale=en-US

Locale for localized copy.

zip
string
Example: zip=28202

Postal/ZIP code for regional eligibility.

effectiveDate
string <date>
Example: effectiveDate=2025-12-19

Date used to evaluate which offers are active.

page
integer >= 1
Default: 1

Page number (1-based).

pageSize
integer [ 1 .. 250 ]
Default: 25

Items per page.
